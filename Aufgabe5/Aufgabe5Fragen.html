Fragen & Antworten

1. Was beschreibt das Interface AssocStringString und wozu kann es verwendet werden?
Das Interface ist ein homogenes assiziatives Array, dem ein Key hinzugefügt wird.

2. Was geschieht in Zeile 9 und was ist das Ergebnis?
Es wird eine Variable port angelegt, die eine process environment für einen Port anlegt, also die Umgebung eines Ports.
Diese Variable port wird in Zeile 10 und 11 auf Nummer 8100 gesetzt.
Mit dem Port als 'Türnummer' kann auf den 'localhost' (der zuvor im Eclipse Terminal gestartet wurde) über den Browser zugegriffen werden.

3. Was geschieht in Zeile 25 von ServerTest.ts?
let query: AssocStringString = Url.parse(_request.url, true).query;
Es wird die Variable 'query' erstellt vom Typ des 'AssocStringString' Interfaces das einen Schlüssel als string erwartet.
Die 'Url.parse()' wandelt einen URL-String um und gibt ein URL-Objekt zurück.
Wenn 'Url.parse()' true ist, dann wird aus query ein neues Objekt vom Interface 'AssocStringString'
Die '_request.url' (Anfrage an Server) kann dann das angefügte 'query' als Objekt (also a und b) ausgegeben.

4. Wie arbeitet die for-in-Schleife in Zeile 29?
Die for-in-Schleife ist für Objekte gedacht, werden alle nicht belegten Array-Stellen einfach übergangen.
Sie iteriert über die Schlüssel des assoziativen Arrays 'AssocStringString'.
Und gibt das angelegte Verzeichnis 'query' aus zu der Schlüssel 'key' passt.

5. Was bewirkt die Header-Information Access-Control-Allow-Origin?
Erlaubt dem Client den Inhalt dieser Seite im Header(Origin) darzustellen.
